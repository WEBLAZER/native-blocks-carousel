# Any Block Carousel Slider

Transform any WordPress block (Query Loop/Post Template, Group, Gallery) into a performant, responsive carousel with **pure CSS**. Zero JavaScript on the frontend.

## 🚀 Try it now (Interactive Demo)

Experience the plugin in seconds with **WordPress Playground**! No installation required:

**[Launch Interactive Demo](https://playground.wordpress.net/?blueprint-url=https://native-blocks-carousel.weblazer.fr/playground/blueprint.json)**

---

## Why Any Block Carousel Slider?

- **Zero JavaScript on the frontend**: GPU-friendly, native scroll performance.
- **WordPress Native**: Works directly on your existing Query Loop, Group, or Gallery blocks.
- **Lightweight**: No external libraries like Swiper or Slick.
- **Performance**: High Lighthouse scores out of the box.

## Installation

1. Download the plugin.
2. Upload it to your WordPress site.
3. Activate it and enjoy!
